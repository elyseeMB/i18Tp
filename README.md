# Internationalisation avec React et TypeScript

Implémentation from scratch d'un système d'internationalisation (i18n) en React et TypeScript.

## Langues supportées

- Français
- Anglais
- Chinois
- Espagnol
- Japonais
- Arabe

## Fonctionnalités

- **Traduction dynamique** : Changement de langue en temps réel
- **Formats de dates** : Adaptation selon la locale
- **Devises** : Affichage automatique de la devise appropriée (€, $, ¥, etc.)
- **Nombres** : Formatage des nombres selon les conventions locales
- **Unités de mesure** : Conversion et affichage des unités (km, °C, Mo, etc.)
- **Pourcentages** : Formatage des pourcentages
- **Animations fluides** : Transitions élégantes lors du changement de langue

## Technologies utilisées

- **React** : Interface utilisateur
- **TypeScript** : Typage statique
- **Framer Motion** : Animations
- **Intl API** : Internationalisation native du navigateur
- **Tailwind CSS** : Stylisation
- **shadcn/ui** : Composants UI
